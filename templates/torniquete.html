Torniquete Automatizado

Control de acceso inteligente integrado con bases de datos escolares.

Arquitectura del Sistema

Diseñamos un sistema electromecánico controlado por microcontroladores. El torniquete valida la identidad del estudiante mediante lectura de las credenciales de la DAE y/o el horario del SAES , consultando la base de datos de la escuela en tiempo real para autorizar o denegar el paso dependiendo de la situación grupal o personal del alumno.

Beneficios Principales

  • Mayor seguridad en el plantel.
  • Control de acceso automático.
  • Flujo de entrada más rápido y ordenado.

Hardware Aplicado

  • • Microcontrolador ESP32
  • • Lectores QR
  • • Computadoras que fungen como servidor y puente de información
  • • Sensores inductivos para validación de cruce

Programas Aplicados

  • • Lenguajes como: Python, HTML, JavaScript, C y C++ aplicado a Arduino
  • • Base de datos MySQL
  • • Página web administrativo
